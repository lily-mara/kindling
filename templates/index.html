Kindling server

This is the root page of a Kindling server. This server generates PNG image data that can be displayed on a Kindle. Below you will find all of the images that this server knows how to generate. You can click on the links to see a preview of what this image will look like on your Kindle, and next to the links you will find a command you can run on your kindle to set it up as auto-refreshing that particular image.
